# siscotproject
 

Basic PHP initial structure SISCOT project, online quotation system, with native PHP, native javascript front-end, using Bootstrap library and a template called Ace Admin Free.

** PHP 7.2
 MariaDB 10.1.37
 Ace Admin Free
 Bootstrap 3.3.7
 Jquery 
 DataTables
 Responsive Datatables **
